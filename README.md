# HR_Analytics
Data Analytics Project showcasing HR Workforce Intelligence analysis using Python, SQL, PowerBI 

HR Analytics & Workforce Intelligence Dashboard
Overview:

This project focuses on analyzing company-level HR data to uncover insights related to workforce composition, salary distribution, attrition, and top employee performance. The project demonstrates an end-to-end data analytics workflow using Python for data preparation, SQL for querying, and Power BI for visualization and reporting.

Dataset:

HR employee dataset containing information such as:
Employee ID
Name, Age, Gender
Department, Job Role
Salary, Experience, Employment Type
Attrition status
The dataset is used for exploratory analysis, cleaning, and business intelligence reporting.

Tools & Technologies:

Python (Pandas, NumPy, Matplotlib)
MySQL (Data storage & SQL queries)
Power BI (Dashboard & visualizations)
Jupyter Notebook

Project Steps:
Exploratory Data Analysis (EDA)
Analyzed data distributions, missing values, and key HR metrics.
Data Cleaning & Preparation
Handled missing values, corrected data types, and created derived columns.

SQL Analysis (MySQL)
Loaded cleaned data into MySQL.
Executed SQL queries for insights such as:
Department-wise employee count
Gender distribution
Highest and top employees by salary
Employment type analysis

Power BI Dashboard Development
Connected Power BI to MySQL.
Built interactive visuals using KPIs, slicers, and DAX measures.

Dashboard Highlights:

Total Employees & Attrition Rate
Department-wise & Gender-wise distribution
Salary analysis and top employee insights
Employment type and experience breakdown
Interactive filters for dynamic analysis

Results & Insights:

Identified departments with high employee concentration.
Highlighted top-performing and highest-paid employees.
Revealed workforce composition based on gender and employment type.
Provided actionable insights to support HR decision-making.

How to Run the Project

1.Open the Jupyter Notebook and run the Python scripts for EDA and data cleaning.
2.Load the cleaned dataset into MySQL.
3.Execute the provided SQL queries.
4.Open the Power BI (.pbix) file and refresh the data connection.

Key Takeaway:

This project demonstrates practical skills in data analysis, SQL querying, and business intelligence reporting, making it suitable for Data Analyst, BI Analyst, and Analytics roles.
